# 自定义Flowable可视化前端

## 功能概述

这是一个基于Vue 3 + TypeScript + Ant Design Vue开发的自定义Flowable工作流可视化前端，包含完整的流程设计、监控和管理功能。

## 核心功能模块

### 1. 流程设计器 (Modeler)
- **拖拽式设计**：支持从左侧元素库拖拽各种流程元素到画布
- **可视化连接**：直观的连线功能，支持流程元素间的连接
- **属性配置**：右侧属性面板，可配置元素的各种属性
- **实时预览**：所见即所得的设计体验
- **模型管理**：保存、部署、验证、下载BPMN模型

### 2. 流程监控大屏 (Monitor)
- **数据统计**：实时展示流程和任务的关键统计数据
- **实例监控**：流程实例运行状态实时监控
- **任务跟踪**：待处理任务列表和状态跟踪
- **可视化图表**：流程分布饼图、任务趋势折线图
- **性能指标**：系统性能和处理效率监控

### 3. 流程管理 (Definition/Instance/Task)
- **流程定义管理**：流程模板的增删改查和部署
- **实例监控**：运行中流程实例的状态跟踪
- **任务处理**：任务的签收、处理、转办等功能

## 技术特点

### 前端技术栈
- **Vue 3 Composition API**：现代化的组件开发模式
- **TypeScript**：强类型支持，提升代码质量
- **Ant Design Vue**：企业级UI组件库
- **ECharts**：丰富的数据可视化图表
- **响应式设计**：适配不同屏幕尺寸

## 使用指南

### 1. 流程设计
1. 进入"流程设计"页面
2. 从左侧元素库拖拽元素到画布
3. 点击元素进行属性配置
4. 使用连接线连接流程元素
5. 保存模型并部署

### 2. 流程监控
1. 进入"流程监控"页面
2. 查看各项统计数据
3. 监控流程实例状态
4. 跟踪任务处理情况
5. 分析性能指标

### 3. 流程管理
1. 在"流程定义"中管理流程模板
2. 在"流程实例"中监控运行状态
3. 在"我的任务"中处理待办任务

## API接口

### 模型相关
```typescript
// 模型管理
getModelList(params)     // 获取模型列表
createModel(data)        // 创建模型
getModel(modelId)        // 获取模型详情
updateModel(modelId, data) // 更新模型
deleteModel(modelId)     // 删除模型

// 模型部署
deployModel(modelId)     // 部署模型
exportModel(modelId)     // 导出BPMN文件
```

### 监控相关
```typescript
// 统计数据
getProcessStatistics()   // 流程统计
getTaskStatistics()      // 任务统计
getPerformanceMetrics()  // 性能指标

// 历史数据
getProcessHistory(days)  // 流程历史
getTaskTrend(days)       // 任务趋势
```

## 部署说明

### 环境要求
- Node.js >= 16.0.0
- npm >= 8.0.0

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 注意事项

1. 确保后端Flowable服务正常运行
2. 配置正确的API接口地址
3. 注意跨域问题的处理
4. 生产环境需要配置HTTPS
5. 建议使用Redis缓存提升性能
